### Using props to customize the text

Modify the text to see it change live:

```js live
<FieldText
  uid="id"
  unit="UNIT"
  updateFormData={(k, v) => alert(`Changed: ${k}, val: ${v}`)}
  value="hello from FieldText"
/>
```
